Class Filter

A Filter object determines of a device registration is to be used for notifications. A Filter contains one or more attributes names that correspond to attribute names found in a registration or registration route. Each attribute name is associated with an array of allowed values. If a registration/route pair given to the passed method does not have the required values, the registration/route will fail to pass the filter.
