Class PostTracker

A PostTracker instance maintains a LIFO size-limited queue that contains Node instances that describe when a notification request was received for processing. New Node instances are added using the 'add' method. The 'track' method looks for cached Node instances and if found logs the duration in seconds between the receipt time of the notification and the initiation time stored in the Node instance.
